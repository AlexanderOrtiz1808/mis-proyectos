# 🚀 Mis Proyectos de Python

¡Hola! Bienvenido a mi repositorio. Aquí estoy documentando mi camino de aprendizaje y evolución como futuro Ingeniero de Software, construyendo proyectos desde cero.

## 📂 Proyectos desarrollados

### 1. Mini Trivia (V1 y V2)
Un juego interactivo de preguntas y respuestas que se juega desde la terminal.
* **Conceptos aplicados:** Manejo de variables, `inputs`, condicionales (`if/elif/else`), Estructuras de Datos (Listas y Tuplas), bucles `for` y limpieza de texto de usuario usando métodos como `.lower()` y `.strip()`.

### 2. Calculadora de Gastos Básica
Una herramienta financiera de terminal para registrar compras, que valida el presupuesto en tiempo real para evitar saldos negativos.
* **Conceptos aplicados:** Bucles infinitos (`while`), conversión de datos (`int()`), operadores matemáticos (`-=`), interrupciones de bucle (`break`) y validación lógica de datos antes de mutar el estado.

### 3. Piedra, Papel o Tijeras vs PC (V1 y V2)
Un juego clásico de consola con modo torneo infinito y sistema de puntuación en vivo.
* **Conceptos aplicados:** Librería `random` para generar elecciones de la computadora, lógica booleana estricta (`and`), bucles infinitos (`while True`), interrupciones (`break`), y manejo de estado con contadores de puntos (`+= 1`).
